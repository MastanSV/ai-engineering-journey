    ### 2. Technical Explanation (Beginner ML Engineer Level)

    Linear Regression is one of the simplest and most important supervised learning algorithms.

    **Goal**: Find the best straight line that models the relationship between input features (**X**) and target variable (**Y**).

    The equation is:
    $$
    y = m \cdot x + c
    $$
    - **y** = predicted value (what we want to predict)
    - **x** = input feature
    - **m** (slope/coefficient) = how much y changes when x increases by 1 unit
    - **c** (intercept/bias) = value of y when x = 0

    **How do we find the "best" line?**

    We use a **cost function** called **Mean Squared Error (MSE)**:
    - For every data point, calculate the vertical distance (error) between actual y and predicted y.
    - Square those errors (so big mistakes hurt more).
    - Take the average.

    The best line is the one with the **smallest possible MSE**.

    **Key Metrics**:
    - **R² (R-squared)**: Tells what percentage of variation in Y is explained by X.  
    (0.0 = bad, 1.0 = perfect)
    - **RMSE**: Error in the same unit as Y (very interpretable)
    - **MAE**: Average absolute error (more robust to outliers)

    ---

    ### 3. Key Concepts You Must Master

    Here are the most important things:

    - **Slope (m)**: Average change in Y for 1 unit change in X
    - **Intercept (c)**: Baseline value of Y when X = 0
    - **Residuals**: Actual Y – Predicted Y (vertical errors)
    - **Cost Function**: Usually MSE (we minimize it)
    - **R²**: How good the model is (higher is better)
    - **Assumptions**: Linearity, Independence, Homoscedasticity, Normality of residuals
    - **Overfitting**: When model performs well on train but poorly on test
    - **Interpretation**: Always say “on average” and “holding other factors constant”

    ### 4. Hands-on Coding (Complete Example)

    Let’s use a **realistic House Price vs Size** example.

    ```python
    import numpy as np
    import matplotlib.pyplot as plt
    from sklearn.linear_model import LinearRegression
    from sklearn.metrics import mean_squared_error, mean_absolute_error, r2_score

    # Sample Data: House Size (sq ft) vs Price (in lakhs)
    X = np.array([800, 1000, 1200, 1500, 1800, 2000, 2500, 3000]).reshape(-1, 1)
    y = np.array([25, 30, 35, 45, 55, 60, 75, 90])   # Price in lakhs

    # ==================== Using sklearn (Easiest way) ====================
    model = LinearRegression()
    model.fit(X, y)

    # Predictions
    y_pred = model.predict(X)

    # Get coefficients
    slope = model.coef_[0]
    intercept = model.intercept_

    print("=== Results using sklearn ===")
    print(f"Slope (m)           : {slope:.2f} lakhs per sq ft")
    print(f"Intercept (c)       : {intercept:.2f} lakhs")
    print(f"R² Score            : {model.score(X, y):.4f} ({model.score(X, y)*100:.1f}%)")

    # Calculate metrics
    mse = mean_squared_error(y, y_pred)
    rmse = np.sqrt(mse)
    mae = mean_absolute_error(y, y_pred)

    print(f"MSE                 : {mse:.2f}")
    print(f"RMSE                : {rmse:.2f} lakhs")
    print(f"MAE                 : {mae:.2f} lakhs")

    # ==================== Visualization ====================
    plt.figure(figsize=(10, 6))
    plt.scatter(X, y, color='blue', label='Actual Data')
    plt.plot(X, y_pred, color='red', linewidth=2, label='Best Fit Line')
    plt.xlabel('House Size (sq ft)')
    plt.ylabel('Price (in lakhs)')
    plt.title('Linear Regression: House Price vs Size')
    plt.legend()
    plt.grid(True, alpha=0.3)
    plt.show()

    # ==================== Interpretation ====================
    print("\nInterpretation:")
    print(f"For every additional 1 sq ft, house price increases by ₹{slope*100000:.0f} on average.")
    ```
