| Phase | Weeks  | Focus                                  | 🚀 Project                              |

## Parallel Tracks

- **Week 5+:** Networking sprint (3 Twitter replies/wk, 1 DM/wk to target engineer)
- **Week 9+:** 1 LeetCode/day
- **Week 13+:** 1 system design case/weekend
- **Week 16+:** Resume/LinkedIn live, practice apps start
- **Week 20+:** Daily 15-min interview drills, real apps to target jobs

## Verification Gates

- End of each phase: project deployed + README + 1 public Twitter thread + teach-back ≥ 7/10
- Week 16: 4 live projects
- Week 20: full mock loop ≥ 6/10
- Week 24: capstone shipped, applying actively
- Week 32: ≥1 offer OR ≥3 onsite-stage interviews

## Project Quality Bar (every project must pass)

1. Live demo on HF Space
2. 3-min Loom walkthrough in README
3. Architecture diagram (Excalidraw / hand-drawn)
4. Real eval numbers
5. Twitter thread explaining it
6. Solves a real problem someone would pay to fix
